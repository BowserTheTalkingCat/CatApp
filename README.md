# Cat Translator

## How to run

1 - Download python: https://www.python.org/downloads/

2 - Install python Dependencies:
```
pip install -r requirements.txt
```

3 - Run Python file:
```
python create_spectrogram.py
```

4 - See output in `spectrogram.png` in the repo
